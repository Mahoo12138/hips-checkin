# 项目介绍

这是个 svelte + typescript + tailwindcss 技术栈项目。

- 单页面程序，无服务端，所有数据均在客户端存储。
- 响应式设计，需要兼容不同的设备屏幕尺寸。
- 多个页面，包括欢迎页，登录页、打卡页等。

## 功能介绍

这是个用于模拟某 App 打卡的项目，用户需要登陆账号，项目通过进行接口请求模拟登录。登录后将 token 信息存储在本地，后续的接口请求将在请求头中携带 token 信息。

用户登录后，项目将模拟用户进行打卡操作，包括打卡时间、地点、参与的项目等。用户可以在页面中查看自己的打卡记录，可以进行打卡操作，包括打卡、重新打卡，补充打开等。
